## Run server site after [client-side](https://github.com/Monwar23/job-task-client)

- git clone
- npm install
- create .env file
- update .env to : 
  - DB_USER=Your Mongodb user
  - DB_PASS=Your Mongodb password
  - ACCESS_TOKEN_SECRET=Your token
- nodemon index.js and run this
